# Project Summary

## Title

AI Code Editor

## One-line description

A full-stack AI-powered coding workspace with authentication, code assistance, and saved session history.

## Key points

- React frontend with Monaco editor
- FastAPI backend for auth and AI actions
- MongoDB for user and session storage
- Gemini integration for completion, bug fixing, and explanation
- Render-ready deployment using Docker

## Suggested GitHub repo description

AI-powered code editor built with React, FastAPI, MongoDB, and Gemini.
