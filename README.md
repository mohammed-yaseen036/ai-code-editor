# AI Code Editor

AI Code Editor is a full-stack coding workspace with authentication, Monaco-based editing, AI-assisted code actions, and saved session history.

## Features

- User registration and login with JWT auth
- Monaco editor with Python, JavaScript, TypeScript, Java, and C++
- AI actions for code completion, bug fixing, and explanation
- MongoDB-backed history of recent AI sessions
- Production-ready single-service deployment on Render

## Stack

- Frontend: React + Vite + Monaco Editor
- Backend: FastAPI
- Database: MongoDB
- AI: Gemini via `google-genai`

## Local Setup

### Backend

1. Open a terminal in `backend`.
2. Create and activate a virtual environment.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Copy `backend/.env.example` to `backend/.env` and fill in:

- `MONGO_URL`
- `DB_NAME`
- `SECRET_KEY`
- `GEMINI_API_KEY`

Optional:

- `GEMINI_MODEL`
- `AI_FALLBACK_MODE`
- `CORS_ORIGINS`

5. Start the API:

```bash
uvicorn main:app --reload
```

### Frontend

1. Open a terminal in `frontend`.
2. Install dependencies:

```bash
npm install
```

3. Start the frontend:

```bash
npm run dev
```

Frontend runs on `http://127.0.0.1:5173` and the backend runs on `http://127.0.0.1:8000`.

## Render Deployment

This repo includes a `render.yaml` Blueprint and a production `Dockerfile`.

### Required Render environment variables

- `MONGO_URL`
- `GEMINI_API_KEY`
- `SECRET_KEY` (auto-generated by Blueprint if you use `render.yaml`)

### Important deployment note

For Render, use MongoDB Atlas or another hosted MongoDB provider. A local MongoDB URL like `mongodb://localhost:27017` will not work on Render.

## Main API Routes

- `POST /auth/register`
- `POST /auth/login`
- `POST /code/complete`
- `POST /code/fix`
- `POST /code/explain`
- `GET /code/history`
- `GET /health`
