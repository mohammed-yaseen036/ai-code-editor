# Frontend

This React app powers the AI Code Editor interface.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`

## Environment

Create a `.env` file in this folder if you want to override the backend URL:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```
